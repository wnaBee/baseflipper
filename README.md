# baseflipper

A small utility for converting between different bases.

use: 
```
python3 baseflipper.py
inputstring [format] [outputformat]  
```
### formats:
command | base
--------- | ---------
-h | hexadecimal
-b | binary
-a | ascii text
-d | decimal
-b64 | base64


### outputformats:
command | base
------- | -------
-2h | hexadecimal
-2b | binary
-2a | ascii text
-2d | decimal
-2b64 | base64
